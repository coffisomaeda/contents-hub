# 3-3. コンテンツ登録機能

## 目的

認証済みユーザーが書籍、ゲーム、映像作品をキーワード検索から登録できるようにする。

## 対象

- 書籍: 楽天ブックス API で検索し、`contents` と `books` に保存
- ゲーム: 楽天ブックス API で検索し、`contents` と `games` に保存
- 映像作品: TMDB API で検索し、`contents` と `videos` に保存
- ユーザー登録状態: `user_contents` に保存

## 登録フロー

1. ユーザーは `/contents/new` を開く
2. メディア種別を選択する
3. キーワード検索を行う
4. 候補を登録する
5. `contents`、メディア固有テーブル、`user_contents` に保存する

## 外部 API

API キーはフェーズ 4 で正式管理するため、3-3 では環境変数がある場合だけ検索を有効化する。

- `RAKUTEN_APP_ID`
- `RAKUTEN_ACCESS_KEY`
- `TMDB_API_KEY`
- `WATCHMODE_API_KEY`

API キーが未設定の場合、該当メディアの検索は利用できない。

## 重複登録

- 書籍は ISBN
- ゲームは JAN
- 映像作品は `tmdb_id` と `media_type`
- 外部 ID がない検索結果は `media_type` と `title`

既存コンテンツがある場合はコンテンツマスタを再作成せず、`user_contents` のみ追加する。
