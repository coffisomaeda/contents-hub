## 注意事項

- Watchmode API の無料枠は月 2,500 リクエスト・3 カ国まで。開発中のリクエスト消費に注意
- 楽天ブックス API は短時間の大量アクセスで一時ブロックされる場合がある
- TMDB の画像 URL は `https://image.tmdb.org/t/p/{size}{path}` で組み立てる
- 各 API キーは `.env` で管理し、Git にコミットしない
